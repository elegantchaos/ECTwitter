Overview

A simple Twitter engine.

This engine exposes fairly raw access to the twitter api, but manages the process of sending requests, and parsing results.

For each twitter call that you make, you supply a block (or target & selector) which the engine calls back when the call succeeds or fails.

Authentication is dealt with via the ECTwitterAuthentication helper class.

Parameters

The twitter method to call.
